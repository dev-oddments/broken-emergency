1. 헬멧 위생문제 해결 - 타슈와 따릉이 그리고 헬멧 착용 의무화라는 요소가 맞물려 문제정의가 깔끔함
2. 헬멧 과 자전거 잠금장치의 결합 - 헬멧 착용 의무화에 대응 가능하면서 헬멧 자체의 도난또한 예방 가능하고 헬멧만으로 잠금장치의 기능을 할 수 있으므로 효율적임
